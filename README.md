# spacecamp-website

## Install
Install [mongoDB](https://www.mongodb.com/).

## Usage

### frontend

#### Install
```
frontend$ npm install
```

#### Build
```
frontend$ npm run build
```

#### Run
```
frontend$ npm run dev
```
> http://localhost:8080

### backend

#### Install
```
backend$ npm install
```

#### Run
```
backend$ npm start
```
or
```
backend$ DEBUG=backend:* npm start
```
> http://localhost:3000
